What It Does

Fully automated
publishing pipeline.

The Daily Audit is an automated, headless pipeline that generates news-style short-form videos using AI and publishes them to multiple platforms via official APIs — no manual steps required.

📰

AI News Script Generation

Uses Google Gemini to generate concise news scripts from curated topics — science, history, biology, physics, and more. Each script is fact-based and ready for voiceover.

🎙️

AI Voiceover & Audio

Azure Cognitive Services speech synthesis generates natural-sounding voiceover audio. Background music and SFX transitions are layered automatically per category.

🎬

Automated Video Assembly

MoviePy composites the voiceover, background visuals, captions, and music into a finished short-form video — no video editing software needed.

📤

Multi-Platform Publishing

Uploads the finished video to TikTok (Content Posting API), YouTube (Data API), and Facebook (Reels API) using official OAuth 2.0 authentication for each platform.

🔐

Secure OAuth Authentication

Uses official OAuth 2.0 flows for all platforms. Tokens are cached locally and refreshed automatically. The pipeline only accesses your own authorized accounts.

⏰

Fully Headless Automation

Once configured, the pipeline runs autonomously via Windows Task Scheduler. Videos are generated and published daily with zero user interaction.

How It Works

From topic to TikTok
in five automated steps.

1

Topic Selection & Script Generation

The pipeline selects a topic from predefined categories (science, history, biology, physics, etc.) and generates a concise, fact-based news script using Google Gemini AI.

2

Voiceover & Audio Production

Azure Cognitive Services converts the script into natural speech audio. Category-appropriate background music and transition SFX are mixed in automatically.

3

Video Assembly & Captioning

MoviePy composites the audio, visuals, and on-screen text captions into a finished short-form video. The output is an MP4 ready for publishing.

4

Multi-Platform Publishing

The finished video is uploaded to TikTok (Content Posting API), YouTube (Data API), and Facebook (Reels API) using official OAuth 2.0 authentication for each platform.

5

Verification & Logging

The pipeline polls each platform's API for publish status and logs the results. On failure, it retries and alerts so no publish is silently lost.
