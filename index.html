Educational Myth-Busting · Automated Pipeline

AI-powered
myth-busting shorts
from script to screen.

The Daily Audit is an automated video pipeline that generates educational myth-busting YouTube Shorts. It writes scripts with Google Gemini, narrates them with Azure neural TTS (strict teacher persona), assembles GPU-rendered video with word-level subtitles, and publishes as private drafts — all in one command.

YouTube Shorts (Private Draft) Facebook Reels (Graph API) Azure Neural TTS + SSML GPU-Accelerated Render

Video Formats

Production Features

Everything the pipeline
actually ships.

🧠

Gemini Script Generation

Google Gemini generates tightly-structured scripts in myth/truth or bizarre format. Word budget: 90–110 words. Smart retry handles 429/503/504 overload with backoff.

Production
🎙️

Azure Neural TTS + SSML Persona

Azure SDK speech synthesis with en-US-AndrewMultilingualNeural voice. SSML persona encodes the "strict teacher" character: sighs, whispers, dramatic pauses, and prosody control. Hard-stop on error — no silent fallback.

Production
📝

Word-Level Kinetic Subtitles

Azure returns millisecond-precise word timestamps. Subtitles are rendered frame-by-frame with 3-line kinetic display at y=1580, 80px line height. Every word appears on-screen — no truncation.

Production
🎨

Color Temperature Tinting

Scene 2 (explanation) gets a warm amber overlay. Scene 3 (truth) gets a cold blue cast. 55-point color temperature swing between scenes for subconscious narrative signaling.

Production
🖼️

Auto Image Sourcing

Wikipedia-scraped images for topic context with AI-generated fallback via Imagen 4.0 and FLUX.2-pro. Scene-specific image assignment with redaction bars for sensitive content.

Production
🎬

GPU-Accelerated Render

MoviePy scene-based compilation with NVIDIA h264_nvenc encoder. CRT retro aesthetic with mascot overlays, starting/ending bumpers, and card rotation every 5–7 seconds. Preset p4, CQ 21.

Production
🖼️

Custom Myth/Truth Thumbnails

Diagonal-split thumbnails in both 16:9 and 9:16 aspect ratios. Myth side vs truth side with non-human mascot (CRT Agent, The Stamp, Agent File) for religious compliance.

Production
🗄️

Topic Deduplication

Semantic fingerprint dedup prevents topic reuse. DB fallback pulls from 44+ curated topics when Gemini is overloaded. Topics are locked after production — no repeats.

Pipeline

From topic to upload
in one command.

A single python main_local.py --type myth runs the full pipeline. No intermediate steps, no manual editing, no file juggling.

1

Topic Selection & Script Generation

The pipeline selects a topic from the fingerprint-deduped database or generates one dynamically. Google Gemini writes a structured script in the chosen format with scene breaks, emotion beats, and SSML voice cues.

Gemini 2.5 Pro · SmartGeminiClient with 504 retry · DB fallback
2

Image Sourcing & Generation

Wikipedia is scraped for topic-relevant images per scene. Missing or failed scrapes fall back to AI image generation (Imagen 4.0, FLUX.2-pro). Images are assigned to specific scenes based on content.

Wikipedia REST API · Imagen 4.0 · FLUX.2-pro · PIL preprocessing
3

Voiceover & Audio Production

Script is split into SSML-tagged scenes. Azure SDK generates neural TTS audio per scene with word-level timestamps. The strict teacher persona is encoded via sighs, whispers, and prosody breaks.

Azure Speech SDK · SSML · en-US-AndrewMultilingualNeural · words.json
4

Video Assembly

MoviePy composites images, audio, bumpers, subtitles, color temperature overlays, CRT filters, and mascot into a 1080×1920 MP4. GPU-accelerated via h264_nvenc. Card rotation every 5–7 seconds with crossfade transitions.

MoviePy 2.1 · h264_nvenc · CompositeVideoClip · ffmpeg concat
5

Thumbnail & Metadata

Diagonal myth/truth thumbnail generated in both 16:9 and 9:16. Title, description, and hashtags auto-generated from topic and format metadata.

PIL · Diagonal composite · auto-hashtag from category tags
6

Upload & Publishing

Video and thumbnail uploaded to YouTube as a private draft via OAuth Data API v3. Facebook Reels published via Graph API. TikTok upload available via Playwright browser automation (Opera GX profile).

YouTube Data API v3 · Facebook Graph API · OAuth 2.0 refresh tokens

⚠️ Current Limitations

YouTube OAuth scope is youtube.upload only — view counts and analytics are not yet accessible. TikTok uses browser automation, not the official API, so it requires an active browser session. The pipeline is run manually via CLI; Windows Task Scheduler automation is not yet configured. No automated status polling after upload — success/failure is logged but not verified against the platform.
